Use case name	ELIMINA SITO DAI PREFERITI
Il Turista elimina un sito dalla lista dei siti preferiti.
Partecipating
Actor	Inizializzato da Turista
Entry
conditions	•	Il Turista si trova nella scheda di un determinato sito.
Flow of events 		User	System
	1.		Sceglie di rimuovere il sito dalla lista dei siti preferiti attraverso la specifica funzionalità.	
	2.			Chiede conferma della rimozione 
	3.		Conferma la rimozione.	
	4.			Rimuove il sito selezionato dalla lista dei siti preferiti.
Exit conditions	•	Il sistema notifica l’avvenuta rimozione del sito dai preferiti.
•	Il Turista annulla l’operazione
•	Interruzione della connessione al server di eTour.

requirements
