Use case name	ELIMINA TAG
Rimuove uno o più tag di ricerca presenti nel sistema.
Partecipating
Actor	Inizializzato da Operatore Agenzia
Entry
conditions	•	L’Operatore Agenzia ha effettuato il login.
Flow of events 		User	System
	1.		Accede alla funzionalità per eliminare un tag.	
	2.			Ricerca nel sistema i tag esistenti e li visualizza in una form.
	3.		Seleziona uno o più tag dall’elenco e invia la richiesta di eliminazione.	
	4.			Elimina i tag di ricerca selezionati.
Exit conditions	•	Il sistema notifica l’avvenuta eliminazione dei tag selezionati.
•	Interruzione della connessione al server di eTour.
