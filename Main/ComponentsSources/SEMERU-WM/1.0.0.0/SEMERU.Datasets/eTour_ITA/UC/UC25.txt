Use case name	VISUALIZZA REPORT STATISTICI
Visualizza i report statistici relativi a una selezionata località.
Partecipating
Actor	Inizializzato da Operatore Agenzia
Entry
conditions	•	L’Operatore Agenzia ha effettuato il login.
Flow of events 		User	System
	1.		Attiva la funzionalità relativa ai report statistici.	
	2.			Carica la lista delle località presenti nel sistema e le visualizza in un form.
	3.		Seleziona una località e sottomette la form.	
	4.			Carica le medie feedback dei siti ottenuti attivando il caso d’uso ricerca sito per la località selezionata. Elabora i report statistici.
Exit conditions	•	Il sistema visualizza i report statistici per la località selezionata.
•	Interruzione della connessione al server

requirements
