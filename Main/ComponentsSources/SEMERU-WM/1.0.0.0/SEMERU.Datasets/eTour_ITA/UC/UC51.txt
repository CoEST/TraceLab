Use case name	INSERISCI SITO NEI PREFERITI
Inserisce un sito selezionato nella lista dei siti preferiti.
Partecipating
Actor	Inizializzato da Turista
Entry
conditions	•	Il Turista si trova nella scheda di un determinato sito.
Flow of events 		User	System
	1.		Attiva la funzionalità per l’inserimento del sito selezionato tra i siti preferiti.	
	2.			Chiede conferma dell’inserimento
	3.		Conferma l’inserimento.	
	4.			Inserisce il sito selezionato nella lista dei siti preferiti.
Exit conditions	•	Il sistema notifica l’avvenuto inserimento del sito nei preferiti
•	Interruzione connessione al server di eTour.
